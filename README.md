# Universo do Futebol ⚽🔥  
Um site dinâmico para acompanhar as últimas notícias e placares de futebol!  

## 🚀 Recursos  
- Notícias atualizadas sobre futebol  
- Acesso rápido aos placares de jogos no Google  
- Interface moderna e responsiva  

## 📌 Como usar  
1. Acesse [Universo do Futebol](https://futebolarte.com)
2. Clique nos botões do menu para acessar notícias ou ver placares de jogos  
3. Aproveite as atualizações em tempo real!  

## 🔧 Tecnologias usadas  
- HTML5  
- CSS3  
- JavaScript
